# Sampling Methods using Seaborn Tips Dataset

This project demonstrates three sampling methods in statistics using Python.

Dataset: seaborn tips

Methods:
1. Simple Random Sampling
2. Stratified Sampling
3. Systematic Sampling

Language: Python
Libraries: seaborn, pandas, numpy


# Seaborn Tips Sampling Project

### 분석 내용
1. `tip` 변수의 분포 및 통계량 분석
2. 평균의 함정과 통계적 특성 설명
3. 3가지 표본 추출(Simple, Stratified, Systematic) 비교 분석

### 기술 스택
- Python, Pandas, Seaborn, Scikit-learn, Matplotlib
